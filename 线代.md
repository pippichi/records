# 二次型

任意一个n元二次型矩阵A，一定存在正交变换x = Qy，将其化为标准型B

![image-20250803220158876](线代.assets/image-20250803220158876.png)

且B的平方和系数就是A的特征值：

![image-20250803221104721](线代.assets/image-20250803221104721.png)

总结：

1、要求正交变换的标准型，其实就是求特征值（特征值就是平方和系数）；

2、要求正交变换x=Qy中的Q矩阵，其实就是求特征向量；
