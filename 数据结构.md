当知道完全二叉树结点个数时，可以推出n0、n1、n2的个数

重点：n0+n2=2*n2+1（n0+n2一定是奇数）

![image-20260416103049500](数据结构.assets/image-20260416103049500.png)



对于有n个结点的二叉树来说，空链域个数为n+1

推理：

1、n个结点中除了根节点以外都有连向父节点的1条线，所以实际有n-1条线；

2、n个结点最多有2n条线（包含连向空链域的线）；

因此，空链域（空指针）应该有`2n-(n-1)=n+1`条线。

## B+树

定义

![image-20260424155830439](数据结构.assets/image-20260424155830439.png)

| m阶B+树                                                      | m阶B树                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 结点的子树个数与关键字个数相等，m个关键字对应m个子树         | m-1个关键字对应m个子树                                       |
| 所有叶子结点包含全部关键字，比如学号。非叶子结点出现的关键字也会出现在叶子结点中 | 各结点关键字不重复                                           |
| 所有分支结点中仅包含它的各个子结点中关键字最大值及指向其子结点的指针，仅起到索引作用，而叶子结点才包含关键字对应记录的存储地址 | 所有结点中都包含了关键字对应记录的存储地址                   |
| 根结点关键字数n∈[1,m]，其他结点关键字数n∈[m/2向上取整,m]     | 根结点关键字数n∈[1,m-1]，其他结点关键字数n∈[m/2向上取整-1,m-1] |
| 有P指针可以顺序遍历所有叶子结点                              |                                                              |
